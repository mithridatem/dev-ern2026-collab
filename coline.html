Dark mode toggle

Créer une fonctionnalité pour switch en mode dark et en mode light

Thème : Dark mode

Date : consulté le 22/05/2026

Auteur / Source : Thomas Steiner, Google

Description :

Un élément personnalisé qui permet d'ajouter facilement un bouton de mode sombre 🌒 sur votre site, afin que vous puissiez initialement respecter les préférences de vos utilisateurs selon prefers-color-scheme, mais aussi leur permettre (éventuellement de manière permanente) de remplacer leur paramètre système uniquement pour votre site.
