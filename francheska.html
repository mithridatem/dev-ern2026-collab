Fiche de veille

Metabase

Un outil open source de Business Intelligence et d'analyse embarquée qui permet de travailler avec des données via des dashboards et des visualisations. Idéal pour créer des rapports visuels à partie d'une base de données.

Thème : Visualisation de données

Date : 22/05/2026 (Le projet existe depuis 2014)

Auteur / Source : Metabase (plusieurs contributeurs)

Description :

Metabase est un outil open source de Business Intelligence (BI) et d'analyse embarquée, développé et maintenu par Metabase, Inc.

Metabase propose un éditeur SQL avancé pour les utilisateurs plus techniques souhaitant écrire des requêtes complexes.

Il est possible d'intégrer des graphiques et dashboards directement dans une application tierce grâce à l'Embedded Analytics SDK

Il permet de créer des dashboards interactifs avec filtres, rafraîchissement automatique et comportements personnalisés au clic.

L'outil supporte de nombreuses bases de données, telles que : MySQL, PostgreSQL, SQLite, Oracle, SQL Server

Le projet est principalement écrit en Clojure (50,9%) pour le backend, et en TypeScript (39,4%) pour le frontend.

Analyse :

Il permet de visualiser rapidement les données stockées en PostgreSQL ou MySQL sans développement supplémentaire, ce qui facilite la présentation et la validation des données.

Intéressant pour tout développeur souhaitant ajoutant une partie BI à son projet.
